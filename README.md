# Кратка информация
Quizzi е мобилно приложение за любознателни хора, които се интересуват от космоса. Въпросникът съдържа въпроси за различни космически обекти, експедиции, астронавти. Всеки ден потребителите получават интересен факт, а към всеки има въпрос има и възможност за повече информация за най-любопитните.
